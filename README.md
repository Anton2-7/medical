# Medical

A simple React web app for working with medical data. It uses React Router for navigation and a custom global hook for state management. Data is loaded from a local `user.json` file. I used a component-based approach when constructing the structure. The project can be optimized by compressing PNG files. I didn't have time to implement hover and active controls on the buttons, but I could certainly work on the visuals if I wanted‚Äîit's just a matter of time.
